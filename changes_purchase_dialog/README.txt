Carpeta: changes_purchase_dialog

Propósito:
Esta carpeta contiene documentación y archivos que resumen los cambios implementados para el diálogo de compra y la ventana de confirmación con enlace a WhatsApp.

Archivos incluidos:
- summary.txt          -> Descripción detallada de lo que se implementó.
- files_list.txt       -> Lista de archivos añadidos y modificados con rutas.
- commit_message.txt   -> Mensaje de commit sugerido para usar en git.
- instructions.txt     -> Comandos y pasos para probar y commitear localmente.

Revisa los archivos y, si quieres, puedo parametrizar el número de WhatsApp en `strings.xml` o añadir más agentes.
